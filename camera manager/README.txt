# Camera Manager v1.0.0
**Developed by Bibek Chand Sah**

## Description
Camera Manager is a Windows application that helps you detect and manage camera devices on your system. You can enable or disable cameras easily through both the main interface and system tray.

## Features
- ✅ Auto-detect integrated and USB cameras
- ✅ Enable/Disable cameras individually 
- ✅ System tray integration with quick controls
- ✅ Real-time status monitoring
- ✅ Auto-administrator elevation

## System Requirements
- Windows 10/11 (64-bit recommended)
- .NET 9.0 Runtime (download from Microsoft if needed)
- Administrator privileges (automatically requested)

## How to Use
1. Right-click CameraManager.exe → "Run as administrator"
2. The application will auto-elevate and show detected cameras
3. Use buttons to Enable/Disable cameras
4. Access quick controls from system tray icon
5. Minimize to tray for background operation

## Controls
- **Refresh**: Scan for camera devices
- **Enable**: Activate selected camera
- **Disable**: Deactivate selected camera
- **Info (ℹ)**: Show application information
- **System Tray**: Right-click for camera controls

## Support
If you encounter any issues:
1. Ensure you're running as Administrator
2. Check Windows Device Manager for camera status
3. Restart the application if cameras don't appear

**Note:** This application modifies system device states and requires administrator privileges for proper camera management.
